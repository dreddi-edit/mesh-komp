# Plan 29-01: Terminal Frontend Fixes — Theme, Copy, Resize

## Objective

Fix the three client-side terminal issues in `assets/app-workspace.js`: (1) update the xterm.js theme so text is legible, (2) add smart Cmd+C copy behavior, (3) wire ResizeObserver for auto-reflow. All changes are confined to the `openTerminal()` and `closeTerminal()` functions.

## must_haves

- Terminal text uses foreground `#c8e6f0` (teal accent) on background `#1a1a1a`
- Cursor color is `#00d4ff`
- Cmd+C with selected text copies to clipboard; Cmd+C with no selection sends SIGINT
- Click-drag text selection works (xterm default, no extra config needed)
- Terminal reflows correctly when its container is resized (ResizeObserver + FitAddon)
- `closeTerminal()` disconnects the ResizeObserver to prevent memory leaks

<read_first>
- `assets/app-workspace.js` lines 1426–1464 (inner try block of openTerminal)
- `.planning/phases/29-terminal-visibility-copy-local-connection/29-RESEARCH.md` §2 (TERM-02 API details)

<action>
In `assets/app-workspace.js`, inside `openTerminal()`, after the line `S.term.open(mountEl);` (approximately line 1441) and before the `setTimeout(()=>S.termFit?.fit(),100)` call, add the following key handler:

```javascript
      S.term.attachCustomKeyEventHandler((e) => {
        if ((e.metaKey || e.ctrlKey) && e.key === 'c' && e.type === 'keydown' && S.term.hasSelection()) {
          const selected = S.term.getSelection();
          if (navigator.clipboard?.writeText) {
            navigator.clipboard.writeText(selected).catch(() => {
              const ta = document.createElement('textarea');
              ta.value = selected;
              ta.style.cssText = 'position:fixed;opacity:0;pointer-events:none';
              document.body.appendChild(ta);
              ta.select();
              document.execCommand('copy');
              document.body.removeChild(ta);
            });
          } else {
            const ta = document.createElement('textarea');
            ta.value = selected;
            ta.style.cssText = 'position:fixed;opacity:0;pointer-events:none';
            document.body.appendChild(ta);
            ta.select();
            document.execCommand('copy');
            document.body.removeChild(ta);
          }
          return false;
        }
        return true;
      });
```

Place this block immediately after `S.term.open(mountEl);` and before `S.termMountSelector = mountSelector;`.
</action>
