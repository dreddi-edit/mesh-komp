# Plan 29-02: Terminal CSS — Container Height and Background

## Objective

Ensure the terminal containers (`#termContainer` in the bottom panel and `#terminalSurfacePrimary` in the terminal surface) have the correct CSS for xterm.js to render visibly — correct background color, flex layout, and minimum height. This is a parallel plan to 29-01 that handles CSS/HTML rather than JS.

## must_haves

- `.term-body` (bottom panel terminal container) has `background: #0d1820` to match the updated xterm theme
- `#terminalSurfacePrimary` `.terminal-surface-body` has `background: #0d1820`
- Both containers have enough height for xterm to render (flex:1 or explicit min-height)
- xterm CSS stylesheet loaded before terminal opens (already via CDN — verify it's present)

---

## Task 29-02-01: Update terminal container background colors in CSS

<read_first>
- `assets/app-workspace.css` lines 200–235 (terminal surface CSS block)
- `assets/app-workspace.css` lines 320–330 (.term-body CSS)
</read_first>

<action>
In `assets/app-workspace.css`, make the following changes:

**1. Update `.terminal-surface` background gradient** (lines ~209–211):

The `.terminal-surface` radial gradient uses `var(--bg)` and `var(--bg5)`. Change so the terminal background specifically pulls toward the dark navy used in the xterm theme. Update `.terminal-surface-body` to have an explicit dark background:

Find:
```css
.terminal-surface-body{flex:1;min-height:0;padding:4px}
```

Replace with:
```css
.terminal-surface-body{flex:1;min-height:0;padding:4px;background:#0d1820;border-radius:8px}
```

**2. Update `.term-body`** (bottom panel terminal container, line ~326):

Find:
```css
.term-body{flex:1;padding:2px;overflow:hidden}
```

Replace with:
```css
.term-body{flex:1;padding:2px;overflow:hidden;background:#0d1820;min-height:120px}
```

**3. Update `.terminal-pane`** to use the dark background on the main pane:

Find:
```css
.terminal-pane{display:flex;flex-direction:column;min-height:0;border:1px solid var(--bd);border-radius:16px;background:rgba(255,255,255,.02);overflow:hidden}
```

Replace with:
```css
.terminal-pane{display:flex;flex-direction:column;min-height:0;border:1px solid rgba(0,212,255,0.15);border-radius:16px;background:#0d1820;overflow:hidden}
```

(Border accent changed to teal to match the new terminal color scheme.)
</action>
