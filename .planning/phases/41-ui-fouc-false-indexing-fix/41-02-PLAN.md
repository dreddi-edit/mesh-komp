# Plan 41-02: Explicit Idle State for Indexing Indicator on Init

**Objective:** Eliminate the false-positive "Indexing..." indicator in the status bar by calling `updateIndexProgressState('idle')` at the start of the `init()` function in `assets/app-workspace.js`. This resets the DOM to hidden state on every page load, before any folder restoration or indexing can begin, covering both the fresh-load-with-no-folder case and the stuck-after-completion case.

**Wave:** 1 (autonomous — single JS line addition, no dependencies)
