# Plan 27-01: OpenAPI Documentation

<objective>
Create OpenAPI/Swagger specification for all /api/* routes with request/response schemas. Serve Swagger UI at /api/docs.
</objective>

<tasks>

<task id="1" name="Create OpenAPI spec">
<read_first>
- src/routes/auth.routes.js
- src/routes/app.routes.js
- src/routes/assistant-workspace.routes.js
- src/routes/assistant-chat.routes.js
- src/routes/assistant-git.routes.js
</read_first>
<action>
Create `src/api-docs/openapi.yaml` with OpenAPI 3.0 specification:

1. Info block: title "Mesh API", version from package.json
2. Servers: `http://localhost:3000`
3. Security schemes: cookie session, CSRF token
4. Paths — document every `/api/*` route:
   - Auth: POST /api/login, POST /api/register, POST /api/logout, GET /api/session
   - Workspace: GET /api/workspace/files, POST /api/workspace/select, GET/PUT /api/workspace/file
   - Chat: POST /api/chat, GET /api/chat/stream (SSE), POST /api/chat/codec
   - Git: GET /api/git/status, POST /api/git/commit, POST /api/git/push, GET /api/git/log
5. Components/schemas — derive from Phase 20 Zod schemas where possible:
   - Request bodies with required/optional fields
   - Response schemas: `{ ok: true, data: {...} }` and `{ ok: false, error: "..." }`
   - Reusable schemas: User, Workspace, File, ChatMessage, GitStatus
6. Error responses: 400 (validation), 401 (auth), 404 (not found), 500 (internal)
</action>
<acceptance_criteria>
- File `src/api-docs/openapi.yaml` exists
- `grep "openapi: 3" src/api-docs/openapi.yaml` matches
- `grep "/api/login" src/api-docs/openapi.yaml` matches
- `grep "/api/workspace" src/api-docs/openapi.yaml` matches
- `grep "/api/chat" src/api-docs/openapi.yaml` matches
- `grep "/api/git" src/api-docs/openapi.yaml` matches
</acceptance_criteria>
</task>

<task id="2" name="Serve Swagger UI">
<read_first>
- src/server.js
- package.json
</read_first>
<action>
1. Run `npm install swagger-ui-express yamljs`
2. Create `src/api-docs/serve.js`:
   ```javascript
   'use strict';
   const swaggerUi = require('swagger-ui-express');
   const YAML = require('yamljs');
   const path = require('node:path');

   function mountApiDocs(app) {
     const spec = YAML.load(path.join(__dirname, 'openapi.yaml'));
     app.use('/api/docs', swaggerUi.serve, swaggerUi.setup(spec, {
       customCss: '.swagger-ui .topbar { display: none }',
       customSiteTitle: 'Mesh API Documentation',
     }));
   }

   module.exports = { mountApiDocs };
   ```
3. In `src/server.js`, mount: `require('./api-docs/serve').mountApiDocs(app);`
</action>
<acceptance_criteria>
- `grep "swagger-ui-express" package.json` matches
- File `src/api-docs/serve.js` exists
- `grep "mountApiDocs" src/server.js` matches
- Navigating to `/api/docs` shows Swagger UI
</acceptance_criteria>
</task>
