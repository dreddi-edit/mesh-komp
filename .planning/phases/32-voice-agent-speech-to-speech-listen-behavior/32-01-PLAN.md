# Plan 32-01: Audio Playback Fix

## Goal

Polly synthesizes speech and sends PCM16 delta chunks to the browser, but no audio is heard. The `AudioContext` is in `suspended` state — browsers require an explicit `resume()` even when the context was created inside a user-gesture handler. Fix by adding `await audioCtx.resume()` in `startAudio()` and a defensive fire-and-forget guard in `playAudioDelta()`.

## Files to Read Before Implementing

- `assets/features/voice-chat.js` lines 530–552 (`startAudio`)
- `assets/features/voice-chat.js` lines 728–734 (`playAudioDelta`)

## Tasks

### Task 1 — Add `audioCtx.resume()` in `startAudio()`

**File:** `assets/features/voice-chat.js`
**Location:** Line 531, immediately after `new AudioContext(...)` assignment

**Current code (line 531):**
```js
audioCtx = new AudioContext({ sampleRate: SAMPLE_RATE });
```

**Replace with:**
```js
audioCtx = new AudioContext({ sampleRate: SAMPLE_RATE });
await audioCtx.resume();
```

### Task 2 — Add defensive resume guard in `playAudioDelta()`

**File:** `assets/features/voice-chat.js`
**Location:** Line 729, after the null-check guard

**Current code (lines 728–734):**
```js
function playAudioDelta(base64) {
  if (!speakerNode || !base64) return;
  const pcm16 = base64ToInt16(base64);
  const float32 = new Float32Array(pcm16.length);
  for (let i = 0; i < pcm16.length; i++) float32[i] = pcm16[i] / (pcm16[i] < 0 ? 0x8000 : 0x7FFF);
  speakerNode.port.postMessage({ type: 'audio-data', float32: float32.buffer }, [float32.buffer]);
}
```

**Replace with:**
```js
function playAudioDelta(base64) {
  if (!speakerNode || !base64) return;
  if (audioCtx && audioCtx.state !== 'running') audioCtx.resume().catch(() => {});
  const pcm16 = base64ToInt16(base64);
  const float32 = new Float32Array(pcm16.length);
  for (let i = 0; i < pcm16.length; i++) float32[i] = pcm16[i] / (pcm16[i] < 0 ? 0x8000 : 0x7FFF);
  speakerNode.port.postMessage({ type: 'audio-data', float32: float32.buffer }, [float32.buffer]);
}
```

## Verification Criteria

- `audioCtx.state !== 'running'` guard appears in `playAudioDelta()`
- No other changes to audio pipeline
