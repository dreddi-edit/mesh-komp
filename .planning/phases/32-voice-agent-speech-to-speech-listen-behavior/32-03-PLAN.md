# Plan 32-03: Stop/Mute During Playback

## Goal

When the user closes/stops the voice orb or taps a mute action during playback, audio output stops without mid-word truncation. Since Polly synthesizes complete sentences before sending, each `streamSpeechResponse` call already represents a full sentence. The "finish sentence" requirement is automatically satisfied — the current sentence's PCM is already fully sent and buffered in the `SpeakerProcessor` worklet queue. Stopping new delta delivery is sufficient.

Implementation: a `muteSpeaker` module-level flag. When set, `playAudioDelta()` returns early (drops incoming delta), the `SpeakerProcessor` drains its existing queue naturally (finishes the current sentence), and the orb transitions to `ready`.

### Task 2 — Apply `muteSpeaker` guard in `playAudioDelta()`

**File:** `assets/features/voice-chat.js`
**Location:** `playAudioDelta()`, after the existing null-check (after Plan 32-01 lands, after the `audioCtx.resume()` guard).

**Current code (after Plan 32-01):**
```js
function playAudioDelta(base64) {
  if (!speakerNode || !base64) return;
  if (audioCtx && audioCtx.state !== 'running') audioCtx.resume().catch(() => {});
  const pcm16 = base64ToInt16(base64);
  const float32 = new Float32Array(pcm16.length);
  for (let i = 0; i < pcm16.length; i++) float32[i] = pcm16[i] / (pcm16[i] < 0 ? 0x8000 : 0x7FFF);
  speakerNode.port.postMessage({ type: 'audio-data', float32: float32.buffer }, [float32.buffer]);
}
```

**Replace with:**
```js
function playAudioDelta(base64) {
  if (!speakerNode || !base64) return;
  if (muteSpeaker) return;
  if (audioCtx && audioCtx.state !== 'running') audioCtx.resume().catch(() => {});
  const pcm16 = base64ToInt16(base64);
  const float32 = new Float32Array(pcm16.length);
  for (let i = 0; i < pcm16.length; i++) float32[i] = pcm16[i] / (pcm16[i] < 0 ? 0x8000 : 0x7FFF);
  speakerNode.port.postMessage({ type: 'audio-data', float32: float32.buffer }, [float32.buffer]);
}
```

```js

    } else {
      stop();
    }
  });
```
