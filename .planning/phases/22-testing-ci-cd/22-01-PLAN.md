# Plan 22-01: CI Pipeline + Coverage

<objective>
Create GitHub Actions CI pipeline with lint, test, coverage (c8), and npm audit. Establishes the safety net before writing new tests.
</objective>

<tasks>

<task id="1" name="Install c8 for coverage">
<read_first>
- package.json
</read_first>
<action>
Run `npm install --save-dev c8` for code coverage.

Add scripts to package.json:
```json
"test:coverage": "c8 --reporter=text --reporter=lcov node --test --test-force-exit --test-timeout=120000",
"test:ci": "c8 --reporter=text --reporter=lcov --check-coverage --lines 40 node --test --test-force-exit --test-timeout=120000"
```
</action>
<acceptance_criteria>
- `grep "c8" package.json` matches in devDependencies and scripts
- `npm run test:coverage` executes without error
</acceptance_criteria>
</task>

<task id="2" name="Create GitHub Actions CI workflow">
<read_first>
- package.json
</read_first>
<action>
Create `.github/workflows/ci.yml`:

```yaml
name: CI
on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main]

jobs:
  lint-test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        node-version: [20]
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: ${{ matrix.node-version }}
          cache: npm
      - run: npm ci
      - name: Lint
        run: npm run lint
      - name: Test with coverage
        run: npm run test:ci
      - name: Upload coverage
        uses: actions/upload-artifact@v4
        with:
          name: coverage
          path: coverage/

  security-audit:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: npm
      - run: npm ci
      - name: Security audit
        run: npm audit --audit-level=high
```
</action>
<acceptance_criteria>
- File `.github/workflows/ci.yml` exists
- `grep "npm run lint" .github/workflows/ci.yml` matches
- `grep "npm run test:ci" .github/workflows/ci.yml` matches
- `grep "npm audit" .github/workflows/ci.yml` matches
- `grep "audit-level=high" .github/workflows/ci.yml` matches
</acceptance_criteria>
</task>

</tasks>

<verification>

</verification>
