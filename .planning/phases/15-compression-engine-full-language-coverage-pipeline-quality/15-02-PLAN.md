# Plan 02 — Grammar Registration: compression-core.cjs + tree-sitter-worker.cjs

## Objective

Register all 8 new language grammars in both `compression-core.cjs` (main thread) and `tree-sitter-worker.cjs` (worker thread). These two files must stay in sync because worker threads are separate V8 isolates — they do not share module scope with the main thread.

Also extend the `definitionLike` node type list in `buildCodeCapsule` to cover Rust, C++, C#, Java, Ruby, PHP, Kotlin, and Swift node types so the tree walk extracts symbols for all new languages.

## must_haves

- .rs, .cpp, .cc, .h, .hpp, .cs, .java, .rb, .php, .kt, .swift all produce capsuleType "structure"
- buildWorkspaceFileRecord("src/main.rs", rustSource) returns record.fileType === "code/rust"
- Both compression-core.cjs and tree-sitter-worker.cjs have require() calls for all 8 new grammars
- Symbol extraction works for at least one symbol per new language
