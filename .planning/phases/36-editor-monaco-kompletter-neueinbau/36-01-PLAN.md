<task type="auto">
  <name>Task 2: Mount Express static route for /assets/monaco/</name>
  <files>src/server.js</files>
  <read_first>src/server.js</read_first>
  <action>
    In src/server.js, add a static route for Monaco immediately after the existing animejs route at line 246.
    
    Current line 246:
    ```javascript
    app.use('/node_modules/animejs', express.static(path.join(REPO_ROOT, 'node_modules', 'animejs'), STATIC_CACHE));
    ```
    
    Add this line directly after it:
    ```javascript
    app.use('/assets/monaco', express.static(path.join(REPO_ROOT, 'node_modules', 'monaco-editor', 'min'), STATIC_CACHE));
    ```
    
    This serves `node_modules/monaco-editor/min/` at `/assets/monaco/` so that:
    - `/assets/monaco/vs/loader.js` → `node_modules/monaco-editor/min/vs/loader.js`
    - `/assets/monaco/vs/editor/editor.main.js` → worker scripts resolve correctly
    - `/assets/monaco/vs/editor/editor.main.css` → CSS file served
    
    Use STATIC_CACHE (1-day cache, not immutable) since version could change between deploys.
    STATIC_CACHE is already defined at line 241: `{ setHeaders: (res) => res.setHeader('Cache-Control', 'public, max-age=86400') }`
  </action>
  <verify>grep -n "assets/monaco" src/server.js</verify>
  <acceptance_criteria>
    - src/server.js contains `app.use('/assets/monaco', express.static(path.join(REPO_ROOT, 'node_modules', 'monaco-editor', 'min'), STATIC_CACHE))`
    - The route appears after the animejs route (after line 246)
    - STATIC_CACHE is used (not IMMUTABLE_CACHE)
  </acceptance_criteria>
  <done>Express route /assets/monaco/ → node_modules/monaco-editor/min/ mounted with STATIC_CACHE</done>
