<task id="3" name="Refactor validation middleware for Zod">
<read_first>
- src/middleware/validate.js
- src/errors/index.js
</read_first>
<action>
Rewrite `src/middleware/validate.js` to:

1. Import `ValidationError` from `../errors`
2. Export `validate(schema, source = 'body')` — returns Express middleware
3. Middleware calls `schema.safeParse(req[source])`
4. On success: set `req.validated = result.data`, call `next()`
5. On failure: throw `new ValidationError('Validation failed', formatZodErrors(result.error))`
6. `formatZodErrors(zodError)` converts Zod error format to `{ field: message }` object

```javascript
function formatZodErrors(zodError) {
  const fields = {};
  for (const issue of zodError.issues) {
    const path = issue.path.join('.');
    fields[path] = issue.message;
  }
  return fields;
}
```
</action>
<acceptance_criteria>
- `grep "safeParse" src/middleware/validate.js` matches
- `grep "ValidationError" src/middleware/validate.js` matches
- `grep "formatZodErrors" src/middleware/validate.js` matches
</acceptance_criteria>
</task>
