# Plan 20-03: CSP Nonce Enforcement

<objective>
Remove `unsafe-inline` from CSP by generating per-request nonces via helmet. Update all 16 HTML pages to use nonce attributes on inline scripts and styles.
</objective>

<tasks>

<task id="1" name="Configure helmet CSP with nonces">
<read_first>
- src/server.js
</read_first>
<action>
Update the helmet CSP configuration in `src/server.js`:

1. Import `crypto` from `node:crypto`
2. Add a middleware before helmet that generates a nonce:
   ```javascript
   app.use((req, res, next) => {
     res.locals.cspNonce = crypto.randomBytes(16).toString('base64');
     next();
   });
   ```
3. Update helmet CSP directives:
   ```javascript
   contentSecurityPolicy: {
     directives: {
       defaultSrc: ["'self'"],
       scriptSrc: ["'self'", (req, res) => `'nonce-${res.locals.cspNonce}'`],
       styleSrc: ["'self'", (req, res) => `'nonce-${res.locals.cspNonce}'`],
       imgSrc: ["'self'", "data:", "blob:"],
       connectSrc: ["'self'", "wss:", "ws:"],
       fontSrc: ["'self'"],
       objectSrc: ["'none'"],
       frameAncestors: ["'none'"],
     },
   },
   ```
4. Remove `'unsafe-inline'` from both `scriptSrc` and `styleSrc`
</action>
<acceptance_criteria>
- `grep "cspNonce" src/server.js` matches
- `grep "unsafe-inline" src/server.js` does NOT match (removed)
- `grep "nonce-" src/server.js` matches
</acceptance_criteria>
</task>

<task id="2" name="Update HTML serving to inject nonces">
<read_first>
- src/server.js
</read_first>
<action>
Update the `sendHtmlWithHashes()` function (or equivalent HTML serving logic) in `src/server.js` to:

1. After reading the HTML content, replace `<script` with `<script nonce="${res.locals.cspNonce}"` for inline scripts
2. Replace `<style` with `<style nonce="${res.locals.cspNonce}"` for inline styles
3. Use string replacement: `.replace(/<script(?![^>]*\bsrc=)/g, '<script nonce="' + nonce + '"')` to only nonce inline scripts (not external script src tags)
4. External scripts with `src=` attribute need nonces too if CSP is strict — add nonce to ALL script tags

Pattern:
```javascript
html = html.replace(/<script/g, `<script nonce="${nonce}"`);
html = html.replace(/<style/g, `<style nonce="${nonce}"`);
```
</action>
<acceptance_criteria>
- `grep "cspNonce" src/server.js` matches in the HTML serving function
- Server starts and pages load without CSP violation errors in console
- `npm test` passes
</acceptance_criteria>
</task>
