## Objective

Add an abort mechanism to `streaming-chat.js` so the user can stop an in-progress AI response. The send button transforms into a stop button during streaming and restores when done.

## must_haves

- While AI is streaming, `#btnSend` shows a stop icon (filled square)
- Clicking the stop button during streaming immediately stops the stream
- Partial response already rendered stays in the message bubble
- After stopping (or completion), `#btnSend` restores to the send arrow icon
- No regression: normal send/receive still works when stop is not clicked
