<task id="1" name="Split model-providers.js into providers/">
<read_first>
- src/core/model-providers.js
</read_first>
<action>
1. Create directory `src/core/providers/`
2. Extract functions by provider into separate files:

   - `openai.js` — OpenAI-compatible provider functions
   - `gemini.js` — Google Gemini provider functions
   - `bedrock.js` — Bedrock streaming, client singleton
   - `byok.js` — Bring-your-own-key credential routing

4. `index.js` re-exports all public functions from sub-modules
5. Update `src/core/model-providers.js` to be a thin re-export facade:
   ```javascript
   'use strict';
   module.exports = require('./providers');
   ```
   This ensures all existing `require('./core').streamAnthropicNative` paths work.
</action>
