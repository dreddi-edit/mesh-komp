# Plan 25-02: Route Migration + Global State Refactor

<objective>
Migrate all route handlers from calling core directly to using service layer. Refactor global mutable state in src/core/index.js to explicit dependency passing.
</objective>

<tasks>

<task id="1" name="Migrate auth and app routes">
<read_first>
- src/routes/auth.routes.js
- src/routes/app.routes.js
- src/services/auth-service.js
</read_first>
<action>
1. `auth.routes.js`: Replace `core.functionName()` calls with `authService.methodName()`:
   - Use `req.app.locals.services.authService` or receive service via route factory

2. `app.routes.js`: Replace core calls with appropriate service calls

3. Preserve existing `{ ok: true/false }` response format — services return plain data, routes format responses

</task>

<read_first>

<action>
Create `test/concurrent-requests.test.js`:

1. Start a test server
2. Fire 10 concurrent requests to different endpoints
3. Verify no race conditions: all responses are correct, no state corruption
4. Check that workspace operations under concurrent load don't share state

```javascript
const { describe, it } = require('node:test');
const assert = require('node:assert/strict');

describe('Concurrent requests', () => {
  it('given 10 parallel requests, when fired simultaneously, then all respond correctly without state corruption', async () => {
    const promises = Array.from({ length: 10 }, (_, i) =>
      fetch(`http://localhost:${port}/api/health`).then(r => r.json())
    );
    const results = await Promise.all(promises);
    results.forEach(r => assert.strictEqual(r.ok, true));
  });
});
```
