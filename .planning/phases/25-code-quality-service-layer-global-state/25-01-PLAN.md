# Plan 25-01: Service Layer Creation

<objective>
Create service modules for each domain that sit between routes and core. Services encapsulate business logic coordination and receive dependencies via factory pattern.
</objective>

<tasks>

<task id="1" name="Create service module structure">
<read_first>
- src/core/index.js
- src/routes/assistant-workspace.routes.js
- src/routes/auth.routes.js
</read_first>
<action>
Create `src/services/` directory with:

1. `workspace-service.js` — wraps workspace core functions:
   - `createWorkspaceService(deps)` factory accepting `{ workspaceOps, infrastructure, context, config, logger }`
   - Methods: `selectWorkspace()`, `readFile()`, `writeFile()`, `searchFiles()`, `batchOps()`, `reindex()`
   - Each method coordinates core calls and returns clean DTOs

2. `assistant-service.js` — wraps assistant + chat core:
   - `createAssistantService(deps)` factory accepting `{ assistantRuns, modelProviders, context, config, logger }`
   - Methods: `startRun()`, `streamChat()`, `inlineComplete()`, `applyProposal()`

3. `auth-service.js` — wraps auth core:
   - `createAuthService(deps)` factory accepting `{ auth, config, logger }`
   - Methods: `login()`, `register()`, `logout()`, `getSession()`, `getStoredCredentials()`

4. `voice-service.js` — wraps voice agent core:
   - `createVoiceService(deps)` factory accepting `{ voiceAgent, config, logger }`
   - Methods: `startVoiceSession()`, `processAudio()`

5. `index.js` — barrel export:
   ```javascript
   module.exports = {
     createWorkspaceService: require('./workspace-service').createWorkspaceService,
     createAssistantService: require('./assistant-service').createAssistantService,
     createAuthService: require('./auth-service').createAuthService,
     createVoiceService: require('./voice-service').createVoiceService,
   };
   ```

Each service file: `'use strict'`, CommonJS, JSDoc on factory and all methods.
</action>
<acceptance_criteria>
- Directory `src/services/` exists with 5 files
- `grep "createWorkspaceService" src/services/workspace-service.js` matches
- `grep "createAssistantService" src/services/assistant-service.js` matches
- `grep "createAuthService" src/services/auth-service.js` matches
- `grep "createVoiceService" src/services/voice-service.js` matches
- `node -e "require('./src/services')"` doesn't throw
</acceptance_criteria>
</task>

<task id="2" name="Wire services in server.js">
<read_first>
- src/server.js
- src/core/index.js
</read_first>
<action>
In `src/server.js`, after core modules are loaded:

1. Create service instances with explicit dependencies:
   ```javascript
   const services = require('./services');
   const workspaceService = services.createWorkspaceService({ workspaceOps, infrastructure, context, config, logger });
   const assistantService = services.createAssistantService({ assistantRuns, modelProviders, context, config, logger });
   const authService = services.createAuthService({ auth, config, logger });
   const voiceService = services.createVoiceService({ voiceAgent, config, logger });
   ```
2. Make services available to routes (e.g., via `app.locals.services` or pass to route factories)
3. Do NOT refactor routes yet — just make services available. Route migration is task 3.
</action>
<acceptance_criteria>
- `grep "createWorkspaceService" src/server.js` matches
- `grep "services" src/server.js` matches
- `npm test` passes
</acceptance_criteria>
</task>
